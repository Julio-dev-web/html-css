# FLEX
    E uma propriedade aplicada ao flex item
    E a simplificacao de "flex-grow,shrink e basis ", a sintaxe deve ser aplicada nessa sequencia:


>flex: 0 (grow) 1(shrink) 150px(basis);


    config padrao >
    
         flex: 0 1 auto  
         flex: initial

         flex: 0 0 auto
         flex: none

         flex: 1 1 auto
         flex: auto

         flex:3
         flex:3 1 auto
