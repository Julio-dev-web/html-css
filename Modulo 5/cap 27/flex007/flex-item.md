# FLEX - ITEM #
Propriedades aplicaveis aos itens.


    ORDER - todo o elemento que estiver dentro do container vai ter por padrao "Order:0"
    

    ALIGN-SELF - cada item vai ter um alinhamento diferente.Ele vai funcionar no eixo tranversal(align-items). Comportamenro individual.

    align-self:
        AUTO
        FLEX-START
        FLEX-END
        CENTER
        STRETCH
